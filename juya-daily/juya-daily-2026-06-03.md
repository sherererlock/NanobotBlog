## [OpenAI 为 Codex 推出六款业务插件与 Sites 和 Annotations功能](https://openai.com/index/codex-for-every-role-tool-workflow/) `#1`
> **OpenAI**为`Codex`推出多项新功能，将应用拓展至更广泛的知识工作场景。此次更新包含**数据分析**等**六类**专属业务插件，可将内容转化为交互式网站的`Sites`功能，以及支持局部精确修改的`Annotations`功能。这些更新正面向`Business`和`Enterprise`用户推出预览版。

## [Anthropic 发布博客详解 Claude Code dynamic workflows 功能](https://claude.com/blog/a-harness-for-every-task-dynamic-workflows-in-claude-code) `#10`
> **Anthropic** 发布博客详解 `dynamic workflows`。该功能让`Claude`能按需编写多`Agent`协作套件，以结构化方式攻克复杂长任务，并可通过“**ultracode**”触发。

**Anthropic** 发布技术博客详解 `dynamic workflows` 功能。该功能允许 **Claude** 根据任务需求，自行编写 `JavaScript` 文件来生成和协调多个子 `Agent`，构建定制化多 `Agent` 协作套件。

该机制旨在对抗长任务中易出现的代理懒惰、自我偏好偏差和目标漂移等常见失败模式，通过为每个子 `Agent` 分配独立上下文窗口和隔离工作树来保障执行质量。
